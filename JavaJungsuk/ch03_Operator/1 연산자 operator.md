### 2.1 증감 연산자 `++` `--`

| 타입   | 설명                        | 예시       |
| ------ | --------------------------- | ---------- |
| 전위형 | 값이 참조되기 **전에** 증가 | `j = ++i;` |
| 후위형 | 값이 참조된 **후에** 증가   | `j = i++;` |



#### 예시

```java
class OperatorEx1 {
  public static void main(String[] args) {
    int i=5;
    i++;
    System.out.println(i);	//결과 : 6
   	i = 5;
    ++i;
    System.out.println(i);	//결과 : 6
    
    i=5;
    int j =0;
    
    j = ++i;		//전위형
    System.out.print("i="+ i + ", j="+ j );	//결과: i=6, j=6
    i = 5;
    j = 0;
    
    j= i++;			//후위형
    System.out.print("i="+ i + ", j="+ j );	//결과: i=6, j=5
  }
}
```
